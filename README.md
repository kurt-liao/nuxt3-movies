## Prerequisites

- TMDB account
- Generate TMDB api key after create an account

## Steps

- Copy `.env-sample` file and rename to `.env`
- Set your api key to `TMDB_API_KEY`
- Enter command `pnpm install`
- Enter command `pnpm dev`
